# HealthPulse360
HealthCare Analytics Dashboard built with Power BI | 56K patient records | DAX, Drill Through, What-If Parameter, Tooltips.

## Overview
A 6-page healthcare analytics dashboard built with Power BI 
using real healthcare dataset with 56K patient records.

## Features
- KPI Cards (Total Patients, Avg Billing, Length of Stay)
- Drill Through (Medical Condition → Patient Detail)
- Conditional Formatting (Red/Green/Yellow for Test Results)
- Bookmarks + Toggle View (Chart vs Scatter Plot)
- What-If Parameter (Insurance Coverage Simulator)
- Report Page Tooltip (Mini dashboard on hover)
- Slicers for interactive filtering
- Navigation Buttons across all pages
- Power Query (Age Group transformation)

## Pages
1. Executive Overview
2. Patient Analysis
3. Financial Analysis
4. Hospital Performance
5. Patient Detail (Drill Through page)
6. Tooltip Page

## Tools Used
- Power BI Desktop
- DAX Measures
- Power Query
- Healthcare Dataset (56K records)

## DAX Measures
- Total Patients
- Avg Billing Amount
- Total Billing Amount
- Avg Length of Stay
- Critical Cases
- Patient Out of Pocket
